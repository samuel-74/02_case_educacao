Análise de Desempenho Acadêmico de Estudantes 📊
Sobre o Projeto

Este projeto realiza uma análise exploratória de dados para investigar os fatores que influenciam o sucesso acadêmico de estudantes em diferentes disciplinas (Matemática, Leitura e Escrita).

A análise foca em entender como variáveis socioeconômicas e de preparação prévia impactam as notas finais, permitindo identificar padrões que podem auxiliar educadores e instituições a desenvolverem estratégias de suporte mais eficazes.
Objetivo

O principal objetivo é responder a perguntas fundamentais sobre o desempenho estudantil, tais como:

    De que forma o nível de escolaridade dos pais influencia os resultados dos filhos?

    Qual o impacto real de cursos preparatórios para testes nas notas finais?

    Existem disparidades significativas de desempenho ligadas a fatores como gênero ou condições socioeconômicas (almoço escolar)?

Tecnologias Utilizadas

    Python (Linguagem de programação)

    Pandas e NumPy (Manipulação e tratamento de dados)

    Seaborn e Matplotlib (Visualização de dados e análise estatística)
